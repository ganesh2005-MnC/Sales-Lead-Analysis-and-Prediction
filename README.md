# Sales-Lead-Analysis-and-Prediction

## Overview
Analyzed real estate sales lead data from multiple advertising channels to identify conversion patterns, lead quality, and business insights.

## Objectives
- Clean and preprocess sales data
- Analyze lead conversion trends
- Identify high-performing lead sources
- Visualize business metrics
- Build predictive models for lead scoring

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Excel

## Key Insights
- Facebook Ads generated highest lead volume
- Referral leads showed highest conversion rate
- Weekend campaigns performed better
